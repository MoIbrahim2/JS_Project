# Animal Expert Quiz

This is a web-based quiz application built with HTML, CSS, and JavaScript.

## Project Summary

This application provides a platform for teachers and students to manage and take exams.

### Teacher Features:

- Create new exams with a specified name, duration, number of questions, and grade level.
- Add questions to exams, including text, optional images, multiple-choice options, and the correct answer.
- Manage existing exams (update, delete).
- Assign exams to students based on their grade.
- View student results and review their answers for each exam.

### Student Features:

- View and take assigned upcoming exams.
- Answer questions within a timed environment.
- Receive immediate feedback on answers.
- View final score after completing an exam.
